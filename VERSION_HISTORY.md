# Lovely Locs Website Version History

This project now includes a built-in visual version switcher on the website at `#versions`.

Open the site and click **Versions** in the top navigation. You can preview the current look or one of seven rollback-style visual presets. These presets change colors, shadows, and visual density without changing services, prices, policies, or contact details.

## Current Version

- `v0` Today: polished conversion layout with featured services, service guide, portfolio preview, testimonials, and first-time client guidance.
